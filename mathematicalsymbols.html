Appendix A: Symbols of Mathematical Logic

Introduction

Consider the statement "For each human, there exists a friend."

We begin the translation of this statement into mathematically meaningful symbols.

Denote the set of humans by $H$ and the set of friends by $F$. A single human in the set of humans is represented by $h\in H$ ("$h$ in $H$").

Similarly, $f\in F$ means "a friend in the set of friends." Now our statement becomes

"For each $H$, there exists a $F$."

Now we use the "upside-down A" symbol $\forall$ to represent the phrase "for each" (typically read as "for all" though).

"$\forall h\in H$ , there exists a $f$ in $F$."

Now we use the symbol $\exists$ to represent the phrase "there exists." If every human has a unique friend, we write $\exists!$.

$\forall h\in H\text{, }\exists f\in F$

If there's some property that $h$ and $f$ satisfy when put together, we put a colon after the statement and follow it by the property. Say, for example, that combining a human and their friend somehow gives the number $10$.

$\forall h\in H\text{, }\exists f\in F: h+f=10$
